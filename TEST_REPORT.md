## 三、Bug定位与分析

### 3.1 已修复Bug清单

| Bug ID | 严重程度 | 问题描述 | 根本原因 | 修复方案 |
|--------|----------|----------|----------|----------|
| BUG-001 | P0 | ClearAllCommand未定义 | 直接使用类名而非延迟加载 | 使用`_get_ui_components().ClearAllCommand` |
| BUG-002 | P1 | 缓存访问KeyError | LRU缓存淘汰后直接访问 | 使用`.get()`方法安全访问 |
| BUG-003 | P1 | 除零错误 | img_w/img_h为零时计算坐标 | 添加零值检查 |
| BUG-004 | P1 | 未初始化属性 | moved_item/resized_item未初始化 | 在`__init__`中初始化 |
| BUG-005 | P1 | itemChanged信号循环 | setItem触发信号 | 使用`blockSignals()` |
| BUG-006 | P1 | 字段类型不同步 | 下拉框变更未更新region | 连接`currentTextChanged`信号 |
| BUG-007 | P1 | result_table信号误触发 | _refresh_table触发itemChanged | 使用`blockSignals()` |
| BUG-008 | P2 | 参数键名不一致 | auto_contrast vs auto_contrast_applied | 统一键名 |
| BUG-009 | P2 | QTimer未停止 | 组件销毁时定时器继续运行 | 添加closeEvent和析构函数 |

### 3.2 日志抓取方法

# 在main.py中添加日志配置
import logging
